transytion documentation
========================

transytion is a utility library for easily creating and managinig tweens in game engines like Pygame. It started out being heavily inspired by `Flux <https://github.com/rxi/flux>`_ and `HUMP.timer <https://hump.readthedocs.io/en/latest/>`_ from the `Love2d <https://www.love2d.org/>`_ community, but has since evolved to use syntactical features from Python to allow powerfully concise ways of creating tweens.
